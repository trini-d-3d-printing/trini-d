Trini-D Firebase Admin v8

Updates in this version:
- Calculator operational base values are saved to the Firebase cloud database automatically when edited.
- The same calculator parameter values load on any device after Firebase login.
- Generated invoices are now stored in a separate Bills / Invoices database tab.
- Invoices still also create order records, so the Orders and Budget workflow continues to work.

- admin/index.html
- admin/admin.js
- admin/admin.css
- admin/firebase-config.js
