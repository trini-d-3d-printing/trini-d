Trini-D 3D Printing Website

Open index.html in a browser.
Default mode is light. Dark/light mode is saved in localStorage.
Pages: Home, About, Services, Gallery, Quotation, Contact Us.
Accepted files highlighted: .stl and .3mf.

Hero slideshow update:
- The home page now includes an automatic slideshow on the right side of the hero section.
- Slideshow images are stored in /assets/ with names starting hero-slide-...
- To use your own real 3D printed item photos, replace those files with your photos using the same filenames.

Admin Panel Update:
- Admin dashboard added in /admin/.
- Open admin/index.html locally for testing.
- Use README_ADMIN_SETUP.txt before publishing.
- Default admin PIN is 1234. Change it in admin/admin.js.
