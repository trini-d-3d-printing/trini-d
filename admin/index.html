Export

Backup your admin database.

This version saves the admin database to Firebase Cloud Firestore and keeps a local browser cache for backup/offline loading.

Desktop DB import reads trini_d_print_database.sqlite3 exported from the desktop software. After import, records are uploaded to Firebase cloud automatically. Direct SQLite import needs internet once for the browser SQLite reader. Offline fallback: use tools/convert_desktop_db_to_admin_json.py, then import the generated JSON here.
